# WeatherAPI (CLI)

Скрипт показывает текущую погоду по названию города через OpenWeatherMap API.

## Требования

- Python 3.10+.

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Настройка API-ключа

1. Создайте файл `.env` в корне репозитория.
2. Заполните его по образцу `.env.example`, указав `OPENWEATHER_API_KEY`.

## Запуск

```bash
python weather.py Moscow
```

Скрипт выведет в консоль текущую температуру (в градусах Цельсия) и текстовое описание погоды.

## Примеры ошибок

- Если город не найден или API вернул ошибку - скрипт покажет понятное сообщение и не упадет с traceback.
